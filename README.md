# -BMI-Calculator-Tkinter-
A simple **Body Mass Index (BMI) Calculator** built with **Python** and **Tkinter**.  
The application provides a graphical user interface (GUI) where users can calculate their BMI, save results locally, and search previously saved records.

## Features

- 🖥️ User-friendly graphical interface using Tkinter
- 📏 Calculate Body Mass Index (BMI)
- 💾 Save BMI records to a local text file
- 🔍 Search previously saved records by name
- 🖼️ Custom GUI with images and Persian interface
- 📂 Multiple windows (Main window + Memory/Search window)

  ## Technologies Used

- Python 3
- Tkinter
- MessageBox
- File Handling
